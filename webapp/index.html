Для чего нужна авторизация?

Для получения доступа к ценам и функциям бота @Tonnel_Network_bot, нам необходимо привязать ваш аккаунт Telegram. Это позволит:

  • Получать актуальные цены в реальном времени
  • Иметь доступ к эксклюзивным функциям бота
  • Получать уведомления о важных обновлениях

Ваши данные используются исключительно для авторизации и не передаются третьим лицам.

Завершение авторизации...

⚠️ Первая авторизация

Для первой авторизации необходимо использовать Telegram Desktop на компьютере. После этого вы сможете использовать бота с любого устройства.

Пожалуйста:

  1. Откройте Telegram Desktop на компьютере
  2. Перейдите к боту
  3. Повторите процесс авторизации

После успешной авторизации на компьютере, вы сможете использовать бота с телефона.

✅ Авторизация успешна!

Теперь вы можете использовать бота с любого устройства. Доступны следующие команды:

  • /prices - Получить текущие цены
  • /help - Показать список команд
